# VengaNames

VengaNames is a web-based game inspired by the popular game "CodeNames". It allows players to compete in a word-guessing game where they have to guess words associated with their team while avoiding words associated with the opposing teams.

## Features

- Join either Team A, Team B
- Words are displayed on the screen, each associated with a specific team.
- Players can click on a word to guess it.
- Score is updated based on correct and incorrect guesses.
- The game ends when the "End Game" button is clicked.
- Real-time updates of users and their chosen teams using WebSocket.

## Technologies Used

- React.js
- Material-UI
- React Router
- React Hot Toast
- Socket.io

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository: `git clone <repository-url>`
2. Install the dependencies in both the api and app folder: `npm install`
3. Start the development server: `npm start`
4. Start the client side: `npm run dev`
5. Open the application in your browser: `http://localhost:5173/`
